في هذا المشروع تم تنفيذ مخزن دائري (Circular Buffer) باستخدام لغة C بشكل يدوي

الهدف هو فهم طريقة تخزين البيانات بشكل منظم داخل مصفوفة ثابتة الحجم، والتعامل مع القراءة والكتابة بدون الحاجة لإزاحة العناصر كما في المخازن الخطية



اعتمدنا في الحل على الأفكار التالية:

استخدمنا struct يحتوي على:

- مصفوفة لتخزين البيانات
- مؤشر head للقراءة
- مؤشر tail للكتابة
- متغير count لمعرفة عدد العناصر داخل 

تم استخدام معامل باقي القسمة (%) حتى يتم إعادة المؤشرات إلى بداية المصفوفة عند الوصول للنهاية، مما يجعل المخزن يعمل بشكل دائري


تم التحقق من حالتين:

- Overflow: عند محاولة الكتابة في مخزن ممتلئ
- Underflow: عند محاولة القراءة من مخزن فارغ

وتم اختبار البرنامج باستخدام إدخال اسم من المستخدم وإضافة "CE-ESY" إليه

الحالة الأولى:

عند استخدام حجم مخزن مناسب، تم تخزين وقراءة البيانات بشكل صحيح

الحالة الثانية:

عند تقليل حجم المخزن، ظهرت رسالة Buffer Overflow كما هو متوقع
